# <a name="how-to-set-up-a-colored-indicator-on-cues-for-the-company-or-individual-users"></a><span data-ttu-id="95190-103">Практическое руководство. <span data-ttu-id="95190-104">Вы как администратор можете настроить очереди, отображаемые в ролевых центрах пользователей, чтобы включить индикатор, меняющий цвет на основе значений данных в очереди.</span><span class="sxs-lookup"><span data-stu-id="95190-104">As an administrator, you can set up Cues that appear on the users' Role Centers to include an indicator that changes color based on the data values in the Cues.</span></span>

<span data-ttu-id="95190-108">В окне **Настройка очереди** можно настроить индикаторы для всех очередей, доступных в базе данных организации.</span><span class="sxs-lookup"><span data-stu-id="95190-108">From the **Cue Setup** window, you set up indicators for all the Cues that are available in the company database.</span></span> <span data-ttu-id="95190-109">Можно настроить индикаторы для применения ко всем пользователям компании или только к одному пользователю.</span><span class="sxs-lookup"><span data-stu-id="95190-109">You can set up the indicators to apply to all users in the company or an individual user only.</span></span> <span data-ttu-id="95190-110">Параметры индикатора в окне **Настройка очереди** функционируют как параметры индикатора по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="95190-110">The indicator settings in the **Cue Setup** window act as the default indicator settings.</span></span> <span data-ttu-id="95190-111">Если окно **Конечный пользователь настройки очереди** доступно пользователям, они могут настроить параметры индикаторов, определяемые в окне **Настройка очереди**.</span><span class="sxs-lookup"><span data-stu-id="95190-111">If the **Cue Setup End User** window is made available to users, then they can personalize the indicators settings that you define in the **Cue Setup** window.</span></span>  
  
<span data-ttu-id="95190-112">Для настройки индикатора нужно задать до двух пороговых значений, определяющих три диапазона значений данных (низкий, средний и высокий), к которым можно применить другой цвет (или стиль).</span><span class="sxs-lookup"><span data-stu-id="95190-112">To set up the indicator, you specify up to two threshold values that define three ranges of data values (low, middle, and high) to which you can apply a different color (or style).</span></span>

# <a name="walkthrough-conducting-a-sales-campaign"></a>Пошаговое руководство. Проведение кампании по продажам
Кампанией называется любой вид деятельности, в которой задействовано несколько контактных лиц. Важной составляющей ее настройки является выбор целевой аудитории. Для этого в [!INCLUDE[d365fin](includes/d365fin_md.md)] с помощью фильтров необходимо создать сегмент или группу контактных лиц.  

 Эти функции используются в модуле "Продажи и маркетинг" для тщательного планирования маркетинговой деятельности и управления взаимодействием с контактными лицами и клиентами. Создав кампании и настроив сегменты контактов, можно осуществлять рассылки или другие виды взаимодействия с контактными лицами и потенциальными клиентами.  

 Благодаря использованию автоматизированных процессов функции «Кампания» и «Сегмент» позволяют планировать, организовывать и отслеживать маркетинговую деятельность. Это увеличивает шансы на привлечение новых клиентов и сохранение существующих.  

## <a name="about-this-walkthrough"></a>Об этом пошаговом руководстве  
 В этом пошаговом руководстве демонстрируется процесс подведения итогов участия в выставке и выбор потенциальных клиентов (контактных лиц) для проведения последующей кампании.  

 В пошаговом руководстве демонстрируются функции управления кампаниями и сегментами в отделе продаж и маркетинга. В этом пошаговом руководстве рассматриваются следующие задачи:  

-   настройка кампании;  
-   выбор целевой аудитории;  
-   отбор данных;  
-   отправка писем контактным лицам;  
-   регистрация отзывов о кампании.  

## <a name="roles"></a>Роли  
 В этом пошаговом руководстве демонстрируются задачи, выполняемые исполнителями следующих ролей:  

-   Менеджер по маркетингу или менеджер по продажам  
-   Штатный сотрудник отдела маркетинга  

## <a name="prerequisites"></a>Предварительные требования  
 Перед выполнением заданий этого пошагового руководства необходимо установить [!INCLUDE[d365fin](includes/d365fin_md.md)].  

## <a name="story"></a>Сюжет  
 Менеджер по маркетингу в отделе продаж организации CRONUS International Ltd. отвечает за планирование кампаний и их проведение. Он также принимает решения относительно участия в тех или иных выставках и оценивает ход кампании.  

 Штатный сотрудник отдела маркетинга занимается созданием, распространением и размещением маркетинговых материалов.  

 Организация выпустила на рынок новый продукт под названием Millennium Server. Недавно продукт рекламировался на выставке Computer Futurus. Многих клиентов он очень заинтересовал, и с целью рекламы клиентам, которые приобретали Millennium Server в период проведения кампании, предлагалась специальная акционная цена.  

 Одна из задач штатного сотрудника маркетингового отдела после выставки заключается во вводе данных обо всех потенциальных клиентах с целью создания для них контактов.  

 Менеджер по маркетингу настраивает кампанию, создает сегмент, содержащий все новые контакты, а затем собирает контактные данные для выбора целевой аудитории кампании.  

 Штатный сотрудник помогает рассылать письма благодарности всем контактам, оставившим свои карточки сотрудникам у стенда, а в конце менеджер записывает все отзывы, полученные от потенциальных клиентов.  

## <a name="setting-up-a-campaign"></a>Настройка кампании  
 После того, как штатный сотрудник вводит данные визитных карточек, полученных во время выставки, менеджер по маркетингу настраивает карточку кампании для управления входящими в кампанию действиями.  

### <a name="to-set-up-a-campaign"></a>Процедура настройки кампании  

1.  Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Кампании**, а затем выберите связанную ссылку.  
2.  Чтобы создать новую кампанию, выберите действие **Создать**. На карточке кампании нажмите клавишу ВВОД, чтобы программа автоматически вставила номер кампании.  
3.  В поле **Описание** введите описание кампании, например **"Выставка FUTURUS"**.  
4.  Выберите поле **Код статуса** и выберите код статуса из списка, который откроется в окне **Статус кампании**.  
5.  Заполните соответствующим образом поля **Дата начала** и **Дата окончания** кампании.  

## <a name="selecting-the-target-audience"></a>Выбор целевой аудитории  
 Менеджер по маркетингу создает сегмент, чтобы выбрать контакты, с которыми он желает наладить отношения.  

### <a name="to-create-a-segment-with-the-relevant-contacts"></a>Процедура создания сегмента с соответствующими контактами  

1.  Выберите действие **Сегменты**.  
2.  Выберите действие **Создать**, чтобы создать новый сегмент. На карточке сегмента нажмите клавишу ВВОД, чтобы программа автоматически вставила номер сегмента.  
3.  На экспресс-вкладке **Общее** в поле **Описание** введите, например, **Посетители выставки FUTURUS**.  

     После ввода общих сведений о сегменте выберите контакты, которые необходимо в него включить.  

     Для выбора контактов можно использовать множество критериев, например можно выбрать работающие у клиента или потенциального клиента контактные лица, отвечающие в своей организации за покупки.  

     Фильтры используются для добавления контактов по критериям, которые лучше подходят для ваших целей. Например, можно выбрать фильтр по должностным обязанностям контактного лица, деловым отношениям или отрасли, в которой работает организация контакта. Для выполнения этого пошагового руководства для выбора контактов воспользуйтесь фильтром **Должностные обязанности**.  

4.  В окне **Сегмент** выберите действие **Добавить контакты**, чтобы открыть фильтр **Добавить контакты**.  
5.  На экспресс-вкладке **Должностные обязанности** выберите для фильтра **Покупка** значение **Код должностной обязанности** и нажмите кнопку **OK**.  

     Теперь в окне **Сегмент** содержится список контактов, основанный на введенном фильтре. На экспресс-вкладке **Общее** в поле **Число строк** можно просмотреть число контактов, отвечающих этим критериям.  

    > [!NOTE]  
    >  Критерии сегментации можно сохранить для повторного использования на последующем этапе.

    1.  В окне **Сегмент** выберите действие **Сегмент**, затем выберите действие **Сохранить критерии**.  
    2.  В окне **Сохранение критериев сегмента** введите код сегмента. В поле **Описание** введите описание критериев сегмента.
    3.  Нажмите кнопку **ОК**.  

## <a name="mining-the-data"></a>Отбор данных  
 Менеджер по маркетингу подробно изучает сегментированный список и понимает, что он слишком велик. Он решает сократить список, оставив в нем только действительно потенциальных клиентов, чтобы сосредоточиться на правильной целевой группе. Этот процесс уточнения и сокращения также называется отбором данных.  

### <a name="to-remove-contacts-from-the-segment"></a>Процедура удаления контактов из сегмента  

1.  В окне **Сегмент** выберите действие **Контакты**, затем выберите действие **Сократить контакты**, чтобы открыть окно **Удаление контактов - сокращение**.  
2.  На экспресс-вкладке **Деловое отношение** выберите для фильтра **PROS** значение **Код деловых отношений** и нажмите кнопку **OK**.  

     В окне **Сегмент** отобразится сокращенный список контактов, а в поле **Число строк** — число контактов, удовлетворяющих этим новым критериям.  

    > [!NOTE]  
    >  Если необходимо отменить это удаление группы контактов, это можно сделать с помощью функции **Возврат**. Иными словами, можно отменить последнее сегментирование.  
    >   
    >  В окне **Сегмент** выберите действие **Сегмент**, затем выберите действие **Возврат**.  
    >   
    >  Только что удаленные контакты добавляются обратно в список контактов.  

## <a name="linking-a-segment-to-a-campaign"></a>Связывание сегмента с кампанией  
 Менеджер по маркетингу решает, что сокращенный список можно использовать в качестве окончательного списка контактов, которые должны быть вовлечены в кампанию. Поэтому он связывает этот сегмент с кампанией по выставке FUTURUS.  

### <a name="to-link-a-segment-to-the-campaign"></a>Процедура связывания сегмента с кампанией  

1.  В окне **Сегмент** на экспресс\-вкладке **Кампания** выберите поле **Код кампании**, чтобы выбрать кампанию, к которой нужно прикрепить сегмент, например **CP0001**.  
2.  Поскольку этот сегмент является целевым для данной кампании, установите флажок **Цель кампании**.  

## <a name="sending-letters-and-email-messages-to-contacts"></a>Отправка писем и сообщений электронной почты контактным лицам  
 Штатный сотрудник отдела маркетинга помогает менеджеру по маркетингу разослать потенциальным клиентам письма с благодарностью за посещение выставки.  

### <a name="to-use-a-segment-to-send-a-letter-to-a-contact"></a>Использование сегмента для отправки буквы контакту  

1.  Откройте карточку **Сегмент** для описания **Посетители выставки FUTURUS**.  
2.  На экспресс-вкладке **Взаимодействие** в поле **Код шаблона взаимодействия** выберите шаблон делового письма (код **BUS**).  
3.  В поле **Тема (умолч.)** введите, следующий текст примера: **Благодарим Вас за посещение выставки**.  

    > [!NOTE]  
    >  Этот шаблон состоит из нескольких документов-вложений на разных языках. Примером языков может служить английский и датский.  

4.  Выберите поле **Код языка (умолч.)**, чтобы открыть окно **Сегмент - языки взаимодействия**. Выберите код языка, а затем нажмите кнопку **OK**.  
5.  Можно просмотреть документ на выбранном языке. Выберите действие **Вложение**, затем выберите действие **Открыть**.  

     Чтобы отреагировать на сообщение с запросом права доступа для запуска Word, выберите параметр **Разрешить для данного сеанса клиента**.  

     При этом откроется вложенный документ Word, чтобы его можно было проверить. Можно также использовать эту возможность, чтобы изменить и модифицировать письмо. По завершении закройте Word.  

6.  Введите тему письма в поле **Тема** на языке, выбранном для шаблона.  
7.  Выберите действие **Журнал**.
8.  Установите флажок **Отправить вложения** для того, чтобы вложения были напечатаны.  

    1. Установите флажок **Создать контрольный сегмент**.  
    2. Нажмите кнопку **OK** для запуска пакетного задания **Регистрировать сегмент**.  

9. Вложения отправляются. После завершения процесса нажмите кнопку **OK** для отображения сообщения, подтверждающего регистрацию сегмента в журнале.  

     Письма печатаются автоматически, и сегмент регистрируется. Поскольку сегмент был зарегистрирован, он больше не содержится в списке сегментов, а перемещается в список зарегистрированных сегментов. Чтобы увидеть этот список, выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Зарегистрированные сегменты**, затем выберите связанную ссылку.  

10. После того как сегмент зарегистрирован, каждое отправляемое письмо регистрируется как взаимодействие, которое можно просмотреть в журнале.  

     Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журналы взаимодействий**, затем выберите связанную ссылку. Для каждой отправленной буквы существует операция.  

### <a name="to-send-an-email-message-to-a-contact"></a>Отправка сообщения электронной почты контакту  

1.  На экспресс-вкладке **Взаимодействие** в поле **Код шаблона взаимодействия** выберите шаблон делового письма (код **BUS**).  
2.  В поле **Тема (умолч.)** введите, следующий текст примера: **Благодарим Вас за посещение выставки**.  
3.  В поле **Тип корреспонденции** выберите **Эл. почта**.  
4.  Определите языковые параметры как в предыдущей процедуре.  
5.  Выберите действие **Журнал**. Откроется окно **Регистрировать сегмент**.  
6.  Установите флажок **Отправить вложения**, чтобы вложения были отправлены по электронной почте.  
7.  Установите флажок **Создать контрольный сегмент**.  
8.  Нажмите кнопку **ОК**.  

     Письма автоматически отправляются по электронной почте, и сегмент регистрируется. Поскольку сегмент был зарегистрирован, он больше не содержится в списке сегментов, а сохраняется в списке зарегистрированных сегментов. Чтобы увидеть этот список, выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Зарегистрированные сегменты**, затем выберите связанную ссылку.  

## <a name="registering-campaign-responses"></a>Регистрация отзывов о кампании  
 В течение следующих нескольких недель потенциальные клиенты будут отвечать на письмо. Менеджер по маркетингу желает отслеживать ответы и записывать отзывы.  

 Для этого настройте сегмент для контактов, ответивших на письмо.  

### <a name="to-register-campaign-responses"></a>Процедура регистрации отзывов о кампании  

1.  В окне **Сегмент** разверните экспресс-вкладку **Взаимодействие**.  
2.  Выберите поле **Код шаблона взаимодействия**.  

     Шаблон взаимодействия для записи отзывов о кампании отсутствует. Поэтому создайте новый шаблон.  

3.  В окне **Шаблоны взаимодействия** выберите действие **Создать**.  
4.  В поле **Код** введите **ОТВЕТ**, а в поле **Описание** — **Отзывы о кампании**.  
5.  Нажмите кнопку **ОК**.  
6.  Выберите этот шаблон взаимодействия в поле **Код шаблона взаимодействия** и в отобразившемся окне подтвердите необходимость обновления строк сегмента с таким же кодом шаблона взаимодействия.  

     Теперь укажите, что эти контактные лица оставили свои отзывы о кампании.  
7.  На экспресс\-вкладке **Кампания** в поле **Код кампании** выберите кампанию.  
8.  Оставьт поле **Код кампании** и в отобразившемся окне подтвердите необходимость обновления строк сегмента с таким же кодом шаблона взаимодействия.  
9. Выберите поле **Отклик кампании** и дайте утвердительный ответ при отображении следующего сообщения.  

     Зарегистрируйте сегмент, чтобы обеспечить запись взаимодействия.  
10. В окне **Сегмент** выберите действие **Журнал**.  
11. В окне **Сегмент журнала** снимите флажок **Отправить вложения**, а затем нажмите кнопку **OK** и дайте утвердительный ответ при отображении следующего сообщения.  

     После регистрации сегмента для кампании автоматически создается операция для записи данного действия в окне **Операции кампании**.

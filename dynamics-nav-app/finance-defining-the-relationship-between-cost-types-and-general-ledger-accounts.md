# <a name="defining-the-relationship-between-cost-types-and-general-ledger-accounts"></a>Определение отношения между типами затрат и счетами главной книги.
Отношение между типом себестоимости и счетом главной книги создается в типе себестоимости и в счете главной книги.  

* Поле **Диапазон счетов ГК** в таблице **Тип затрат** определяет, какие счета главной книги относятся к типу затрат.  
* Поле **Номер типа затрат** в диаграмме счетов определяет, к какому типу затрат относится счет главной книги.  

Эти 2 поля заполняются автоматически при использовании функции **Извлечь типы затрат из плана счетов**.  

## <a name="relationship-between-general-ledger-accounts-and-cost-types"></a>Взаимосвязь между счетами главной книги и типами себестоимости.  
Существует отношение n:1 между счетами главной книги и типами себестоимости. Несколько счетов Главной книги может принадлежать к одному типу себестоимости, но каждый финансовый счет может принадлежать только к одному типу себестоимости. В следующей таблице описываются подробные сведения в отношении.  

|Связь|**Диапазон счетов ГК**|**Номер типа затрат**|  
|------------------|------------------------------------------------|-------------------------------------------|  
|Один счет Главной книги для каждого типа себестоимости|Один счет Главной книги|Один тип себестоимости|  
|Несколько счетов Главной книги для одного типа себестоимости|Диапазон счетов Главной книги, например, 7110..7193 для каждого счета Главной книги|Для каждого счета Главной книги в диапазоне существует только один тип затрат|  
|Типы затрат без соответствующих счетов Главной книги|<Empty>||  
|Счета Главной книги для данных операций не будут перенесены||<Empty>|  

## <a name="cost-types-without-a-relationship-to-the-general-ledger"></a>Типы затрат без отношения с главной книгой  
Тип затрат не может иметь связи со счетами Главной книги, если истинно одно из следующих условий:  

* Счета для операционного учета, например "Вычисляемый процент и амортизацию", принимают только расходы из операционного учета.  
* Такие типы затрат, как 9902, 9901 и 9903 в базе данных [!INCLUDE[d365fin](includes/d365fin_md.md)], используются как дебетовые и кредитовые счета для распределения.  
* Вспомогательный счет, 9920 в базе данных [!INCLUDE[d365fin](includes/d365fin_md.md)], содержит фактические начисления, которые показывают разницу между себестоимостью и расходами из главной книги.

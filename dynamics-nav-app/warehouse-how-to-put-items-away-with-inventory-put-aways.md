# <a name="how-to-put-items-away-with-inventory-put-aways"></a><span data-ttu-id="33db3-104">Практическое руководство. Размещение товаров с помощью складского размещения</span><span class="sxs-lookup"><span data-stu-id="33db3-104">How to: Put Items Away with Inventory Put-aways</span></span>
<span data-ttu-id="33db3-105">Если настройки склада таковы, что требуется обработка размещения, но не требуется обработка приемки, то, чтобы зарегистрировать и учесть информацию о размещении и приемке для документов-источников, нужно пользоваться документом **Размещение запасов**.</span><span class="sxs-lookup"><span data-stu-id="33db3-105">When your location is set up to require put-away processing but not receive processing, you use the **Inventory Put-away** document to record and post put-away and receipt information for your source documents.</span></span> <span data-ttu-id="33db3-106">Входящим документом-источником может быть заказ покупки, заказ возврата продажи, входящий заказ передвижения, заказ на сборку или производственный заказ, готовая продукция которого готова для размещения.</span><span class="sxs-lookup"><span data-stu-id="33db3-106">The inbound source document can be a purchase order, a sales return order, an inbound transfer order, or an assembly or production order whose output is ready to be put away.</span></span>  

<span data-ttu-id="33db3-107">Размещение запасов можно создать тремя способами:</span><span class="sxs-lookup"><span data-stu-id="33db3-107">You can create an inventory put-away in three ways:</span></span>  

- <span data-ttu-id="33db3-108">Создание размещения в два этапа, при котором сначала создается складской запрос из документа-источника, являющийся для склада сигналом о том, что документ-источник готов для размещения.</span><span class="sxs-lookup"><span data-stu-id="33db3-108">Create the put-away in two steps by first creating a warehouse request from the source document, which acts as a signal to the warehouse that the source document is ready for put-away.</span></span> <span data-ttu-id="33db3-109">После этого складское размещение можно создавать в окне **Размещение запасов** на основании документа-источника.</span><span class="sxs-lookup"><span data-stu-id="33db3-109">The inventory put-away can then be created from the **Inventory Put-away** window based on the source document.</span></span>  
- <span data-ttu-id="33db3-110">Создание складского размещения непосредственно из самого документа-источника.</span><span class="sxs-lookup"><span data-stu-id="33db3-110">Create the inventory put-away directly from the source document itself.</span></span>  
- <span data-ttu-id="33db3-111">Создавайте складские размещения одновременно для нескольких документов-источников с помощью пакетного задания.</span><span class="sxs-lookup"><span data-stu-id="33db3-111">Create inventory put-aways for several source documents at once by using a batch job.</span></span>

    <span data-ttu-id="33db3-117">Для производственных заказов запрос склада создается посредством создания входящего запроса из выпущенного производственного заказа.</span><span class="sxs-lookup"><span data-stu-id="33db3-117">For production orders, you create the warehouse request by creating an inbound request from the released production order.</span></span>  
3.  <span data-ttu-id="33db3-118">Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Запущенные произв. заказы**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="33db3-118">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Released Production Orders**, and then choose the related link.</span></span>  
4. <span data-ttu-id="33db3-119">Выберите действие **Создать вход. складской запрос**.</span><span class="sxs-lookup"><span data-stu-id="33db3-119">Choose the **Create Inbound Whse. Request** action.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="33db3-120">Можно также создать входящий запрос склада, если установить флажок **Создать входящий запрос** при обновлении производственного заказа.</span><span class="sxs-lookup"><span data-stu-id="33db3-120">You can also create the inbound warehouse request by selecting the **Create Inbound Request** check box when you refresh the production order.</span></span> <span data-ttu-id="33db3-121">Дополнительные сведения см. в разделе [Практическое руководство. <span data-ttu-id="33db3-122">После создания складского запроса работник склада, которому поручено размещение, сможет увидеть, что документ-источник готов, и создать документ складского размещения.</span><span class="sxs-lookup"><span data-stu-id="33db3-122">When the warehouse request is created, a warehouse employee assigned to putting items away can see that the source document is ready and can create an inventory put-away document.</span></span>

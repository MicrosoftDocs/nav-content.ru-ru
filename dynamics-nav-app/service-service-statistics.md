# <a name="viewing-service-statistics"></a>Просмотр сервисной статистики
Dynamics NAV может обеспечить определенную статистику, которую можно использовать для анализа сервисных документов и определения качества управления процессами сервиса. Можно анализировать сервисные контракты, товары, предложения, заказы, счета и кредит-ноты, выбрав действия **Статистика**. Для сервисных товаров и контрактов можно также использовать **Анализ тренда сервисного товара** или **Анализ тренда контракта** для просмотра сводки операций книги сервиса для конкретного сервисного товара.   

## <a name="viewing-statistics-for-service-orders"></a>Просмотр статистики для сервисных заказов
Функция статистики сервисного заказа позволяет быстро просмотреть содержимое всего сервисного заказа, подробности о конкретных сервисных строках и информацию, связанную с выставлением счетов, отгрузкой и потреблением, а также балансом клиента.  

Статистические данные отображаются для сервисного заказа в окне **Сервисный заказ - статистика** для соответствующего заказа. Соответствующее окно статистики можно открыть из сервисного заказа. В окне **Сервисные заказы** выберите **Статистика**. На экспресс-вкладках в этом окне отображается такая информация, как количество, сумма, НДС, себестоимость, прибыль и кредитный лимит клиента. Если не указано иное, суммы в этом окне выражены в валюте сервисного заказа.  

### <a name="view-totals-for-a-service-order"></a>Просмотр итогов для сервисного заказа  
Можно просмотреть итоговую сумму по сервисным строкам (с НДС и без), долю НДС, себестоимость и прибыль по сервисным строкам. В окне также отображается информация по конкретному товару, такая как вес, объем и количество упаковок.  

### <a name="view-shipping-information"></a>Просмотр информации по отгрузке  
Можно посмотреть информацию о товарах, ресурсах или себестоимости для отгрузки. Для предоставления информации программа использует значения, указанные в поле **Кол-во для отгрузки** каждой сервисной строки в заказе.  

### <a name="view-order-details"></a>Просмотр сведений о заказе  
Можно просматривать информацию о товарах, времени использования ресурсов и себестоимости для включения в счета и потребления. В следующей таблице описывается эта информация.  

|Столбец | Описанием|  
|------------|---------------------------------------|  
|**Выставление счетов**|Отображает суммы, подлежащие учету как суммы, включенные в счета, которые выставлены по этому сервисному заказу.|  
|**Потребление**|Отображает количество и себестоимость товаров или ресурсов, которые будут учтены как потребленные.|  
|**Всего**|Отображает общие суммы по сервисному заказу как результат сложения сумм в выставленных счетах и сумм использования.|  

### <a name="analyze-service-order-lines"></a>Анализ строк сервисного заказа  
Можно анализировать информацию по типам сервисных строк, включенных в сервисный заказ. Суммы отображаются раздельно для:  

* Товаров  
* Ресурсы  
* Затраты и счета Главной книги  

### <a name="view-customer-information"></a>Просмотр информации о клиенте  
Просматривайте сальдо по счету клиента, а также максимальный кредит клиенту, для которого создан этот сервисный документ.

## <a name="viewing-service-item-statistics"></a>Просмотр статистики сервисного товара
В окне **Статистика сервисного товара** можно просмотреть актуальную информацию по сервисному товару на основе следующих типов операций книг сервиса:  

* Ресурсы  
* Товаров  
* Затраты на сервисное обслуживание  
* Сервисные контракты  
* Итог  

Для каждого типа операции можно просмотреть сумму выставленного счёта, потребление (сумму), сумму затрат, количество, количество по выставленному счету и потребленное количество, сумму прибыли и процент прибыли. Процент прибыли рассчитывается по следующей формуле:  

* (Сумма по выставленным счетам - Потребление (Себестоимость)) x 100 / Сумма по выставленным счетам  

## <a name="using-trendscapes"></a>Использование анализа трендов
Для сервисных товаров и сервисных контрактов окна **Анализ тренда сервисного товара** или **Анализ тренда сервисного контракта** предоставляют прокручиваемую сводную информацию об операциях книги обслуживания за определенный период времени для конкретного сервисного товара или контракта. Для просмотра анализа тренда откройте сервисный товар или сервисный контракт, выберите действие **Статистика**, затем выберите **Анализ тренда**.

При прокручивании списка вычисляются суммы в локальной валюте, соответствующие указанному интервалу времени. Все суммы рассчитываются на основе операций книги сервиса, то есть операций, создаваемых при учете сервисных заказов или сервисных счетов.

Список можно фильтровать, указав включаемые сервисные товары.  

> [!Tip]  
>  Если задан интервал **День**, и необходимо просмотреть длительный период, можно ускорить эту операцию, переключившись на более широкий интервал, например **Квартал**. После обнаружения нужного периода можно переключиться обратно на исходный интервал, чтобы просмотреть более подробную информацию.   

## <a name="viewing-gains-and-losses-on-contracts"></a>Просмотр прибылей и убытков по контрактам  
Операция контрактной прибыли или убытка создается во время преобразования предложения по контракту в сервисный контракт, при добавлении или удалении строк контракта из сервисного контракта и при отмене контракта. Прибыли или убытки по контракту можно просматривать на следующих страницах.  

|Стр. | Описанием|  
|----------------|---------------------------------------|  
|**Прибыли/убытки по контракту (контракты)**|Просмотр прибылей/убытков по контракту по сервисным контрактам|  
|**Прибыли/убытки по контракту (группы)**|Просмотр прибылей/убытков по контракту по группам сервисных контрактов|  
|**Прибыли/убытки по контракту (клиенты)**|Просмотр прибылей/убытков по контракту по клиентам|  
|**Прибыли/убытки по контракту (основание)**|Просмотр прибылей/убытков по контракту по кодам причины|  
|**Прибыли/убытки по контракту (центр отв.)**|Просмотр прибылей/убытков по контракту по центрам ответственности|  

1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите название страницы, которую требуется показать, затем выберите связанную ссылку.  
2. Заполните критерии фильтрации, которые нужно применить. Например, в окне **Прибыли/убытки по контракту (основание)** выберите значение для поля **Фильтр по коду причины**.  
3. Выберите действие **Показать матрицу**.

## <a name="viewing-statistics-for-posted-service-documents"></a>Просмотр статистики для учтенных сервисных документов
Функция сервисной статистики позволяет получить статистический обзор содержимого учтенных сервисных документов, таких как учтенные расходные накладные, учтенные счета и учтенные кредит-ноты.  

Статистическая информация отображается в окне статистики соответствующего учтенного сервисного документа. Соответствующее окно статистики можно открыть из учтенной сервисной расходной накладной, учтенной сервисной накладной или учтенной сервисной кредит-ноты. Для каждого из этих типов документов на вкладке **Главная** в группе **Процесс** выберите **Статистика**. Например, в окне **Отправленные сервисные счета** на вкладке **Главная** в группе **Процесс** выберите **Статистика**.  

### <a name="posted-service-shipment-statistics"></a>Учтенная сервисная расх. накладная - статистика  
В окне **Статистика сервисной расх. накладной** представлен обзор учтенных сервисных расходных накладных. Сюда входит информация о физических параметрах отгрузки, например количестве отгруженных товаров, времени использования ресурсов или себестоимости, а также о весе и объеме отгруженных товаров.  

### <a name="posted-service-invoice-statistics"></a>Учтенная статистика сервисного счета  
В окне **Сервисный счет - статистика** представлена статистическая сводка по учтенному сервисному счету. Можно просматривать итоговые суммы учтенного сервисного счета. Эти данные включают итоговую сумму по сервисным строкам (с НДС и без), которая была учтена как включенная в счет, долю НДС, себестоимость и прибыль по учтенному счету. В окне также отображается приведенная ниже информация:  

* Товары в строках сервисного счета, например вес, объем и количество упаковок.  
* Сальдо по счету клиента, а также максимальный кредит, который вы можете предоставить этому клиенту.  

### <a name="posted-service-credit-memo-statistics"></a>Учтенная сервисная кредит-нота - статистика  
Окно **Сервисная кредит-нота - статистика** можно использовать, чтобы получить статистический обзор строк учтенной сервисной кредит-ноты. Обзор может включать:

* Итоговые суммы по учтенной кредит-ноте, отображаемые как количество, сумма, НДС, себестоимость и прибыль. Также содержится информация о товарах из сервисных строк учтенной кредит-ноты, например количество, вес и объем.  
* Общая информация о клиенте, такая как кредитный лимит клиента и сальдо счета.

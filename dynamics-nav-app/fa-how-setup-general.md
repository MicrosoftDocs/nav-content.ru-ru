# <a name="how-to-set-up-general-fixed-assets-information"></a>Практическое руководство. Настройка общих данных основных средств
Прежде чем управлять основными средствами, необходимо настроить счета ГК по умолчанию, ключи распределения, шаблоны и разделы журналов для учета и переквалификации основных средств; можно также классифицировать основные средства по классам, таким как материальные и нематериальные.

## <a name="to-set-up-general-default-values-for-fixed-assets"></a>Настройка общих значений по умолчанию для основных средств
Общее поведение или функциональность основных средств, а также серии номеров документов настраиваются в окне **Основные средства - настройка**.

1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Основные средства - настройка**, затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля. Выберите поле для чтения краткого описания поля или ссылки на дополнительную информацию.

## <a name="to-set-up-fixed-asset-posting-groups"></a>Настройка учетных групп основных средств  
Для определения групп основных средств используются учетные группы. Операции для этих учетных групп учитываются на тех же счетах ГК.  
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Учетные группы ОС**, затем выберите связанную ссылку.  
2. Выберите действие **Создать**.
3. В окне **Карточка учетной группы ОС** заполните поля, как требуется.

    **Примечания**. Чтобы балансирующие счета для различных видов учета основных средств автоматически вставлялись при выборе действия **Вставить баланс. счет ОС** в строки журнала, выполните следующий шаг на основании учета повышения стоимости.
4. На экспресс-вкладке **Балансирующий счет** в поле **Повышение стоимости - баланс. счет** выберите счет ГК, на котором необходимо учитывать балансовые операции для повышения стоимости.

Дополнительные сведения об использовании действия **Вставить баланс. счет ОС** в строках журнала ГК учета основных средств см., например, раздел [Практическое руководство. Переоценка основных средств](fa-how-revalue.md).

## <a name="to-set-up-fixed-asset-allocation-keys"></a>Настройка ключей распределения основных средств  
Транзакции могут быть распределены по различным подразделениям или проектам в соответствии с определенными пользователем ключами распределения. Например, можно настроить ключи распределения для распределения стоимостей амортизации на машины с 35% для административного подразделения и 65% для торгового подразделения. Дополнительные сведения см. в разделе [Практическое руководство. Использование ключей распределения в финансовых журналах](ui-how-use-allocation-keys-general-journals.md).

Ключи распределения применяются к классам ОС, а не к отдельным основным средствам.  
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Учетные группы ОС**, затем выберите связанную ссылку.  
2. В окне **Учетные группы ОС** выберите действие **Распределения**, затем выберите тип учета.
3. В окне **Распределения ОС** заполните требуемые поля.
4. Повторите шаги 2 и 3 для каждого типа учета, для которого необходимо определить ключи распределения.

## <a name="to-set-up-fixed-asset-journal-templates"></a>Настройка шаблонов журнала основных средств  
Шаблон — это определенный заранее макет журнала. В шаблоне содержится информация о кодах трассировки, отчетах и серийных номерах. Дополнительные сведения см. в разделе [Работа с финансовыми журналами](ui-work-general-journals.md).

Шаблон журнала основных средство создается программой Dynamics NAV автоматически при первом открытии окна **Журнал ОС**, но можно настроить дополнительные шаблоны журнала.  
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Шаблоны журнала ОС**, затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля.

## <a name="to-set-up-fixed-asset-journal-batches"></a>Настройка разделов журнала основных средств
Можно настроить несколько разделов журнала, т. е. индивидуальные журналы для каждого шаблона журнала. Например, у сотрудников могут быть собственные разделы журнала, в которых инициалы сотрудника будут использоваться как название раздела журнала. Дополнительные сведения см. в разделе [Работа с финансовыми журналами](ui-work-general-journals.md).  
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Шаблоны журнала ОС**, затем выберите связанную ссылку.  
2. Выберите соответствующий шаблон журнала, затем выберите действие **Разделы**.
3. В окне **Разделы журнала ОС** заполните требуемые поля.

## <a name="to-set-up-fixed-asset-reclassification-journal-templates"></a>Настройка шаблонов журнала реклассификации ОС  
Для перемещения, разделения и объединения основных средств используются специальные журналы реклассификации. Шаблон журнала реклассификации основных средства создается программой Dynamics NAV автоматически при первом открытии окна **Журнал реклассификации ОС**, но можно настроить дополнительные шаблоны журналов реклассификации ОС. Дополнительные сведения см. в разделе [Работа с финансовыми журналами](ui-work-general-journals.md).  
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Шаблоны журнала реклассификации ОС**, затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля.

## <a name="to-set-up-fixed-asset-reclassification-journal-batches"></a>Настройка разделов журнала реклассификации ОС  
Можно настроить несколько разделов журнала, т. е. индивидуальные журналы для каждого шаблона журнала реклассификации. Например, у сотрудников могут быть собственные разделы журнала реклассификации, в которых инициалы сотрудника будут использоваться как название раздела журнала реклассификации. Дополнительные сведения см. в разделе [Работа с финансовыми журналами](ui-work-general-journals.md).
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Шаблоны журнала реклассификации ОС**, затем выберите связанную ссылку.  
2. Выберите соответствующий шаблон журнала, затем выберите действие **Разделы**.
3. В окне **Разделы журнала реклассиф. ОС** заполните требуемые поля.

## <a name="to-set-up-fixed-asset-class-codes"></a>Настройка кодов классов основных средств  
Коды классов основных средств могут использоваться для разделения основных средств на группы, например, материальные и нематериальные основные средства.
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Классы ОС**, затем выберите связанную ссылку.
2. Введите коды и наименования классов, которые нужно создать.

## <a name="to-set-up-fixed-asset-subclass-codes"></a>Настройка кодов подклассов основных средств
Коды подклассов основных средств можно использовать для разделения основных средств на категории, например здания, транспортные средства, мебель и оборудование.  
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Подклассы ОС**, затем выберите связанную ссылку.
2. Введите коды и наименования классов, которые нужно создать.

## <a name="to-set-up-fixed-asset-location-codes"></a>Настройка кодов местонахождения основных средств
Коды склада основного средства можно использовать для регистрации размещения основных средств, например в подразделениях сбыта, приема, администрации, производства или склада. Эта информация важна при учете в сферах страхования и управления товарами.
1. В правом верхнем углу выберите значок **Поиск страницы или отчета**, введите **Местонахождения ОС**, а затем выберите связанную ссылку.
2. Введите коды и наименования для местонахождения основных средств, которые нужно создать.

## <a name="to-register-opening-entries"></a>Регистрация начальных операций  
Если используете основные средства в Dynamics NAV впервые, необходимо настроить область приложения «Главная книга». Порядок выполнения это зависит от того, какие основные средства интегрированы с Главной книгой.  

 Для учета транзакций основных средств в главной книге используется следующая процедура.  

1. Убедитесь в том, что были выполнены основные процедуры настройки основных средств.  
2. Создайте карточку ОС для каждого основного средства.  
3. Настройте книги амортизации основных средств.  
4. Включите интеграцию главной книги, выполнив следующие шаги.
5. В поле **Поиск** введите **Книги амортизации**, а затем выберите связанную ссылку.  
6. Выберите соответствующую книгу амортизации. На вкладке **Главная** в группе **Управление** выберите **Правка**, чтобы открыть окно **Карточка книги амортизации**.
7. Убедитесь, что на экспресс-вкладке **Интеграция** все поля оставлены незаполненными (снимите все флажки). При наличии нескольких книг амортизации необходимо включить интеграцию ГК для каждой из них.  
8. В журнал основных средств введите следующие строки для каждого основного средства.
    - Строка со стоимостью покупки.
    - Строка с накопленной амортизацией до конца предыдущего финансового года.
    - Строка с накопленной амортизацией с начала текущего финансового года до даты, на которую в Dynamics NAV задано начало расчета амортизации.

При наличии других входящее сальдо, можно также ввести их сейчас, например "Понижение стоимости" и "Повышение стоимости".  

Для учета средств, не интегрированных с главной книгой, пропустите шаги с 4 по 7.

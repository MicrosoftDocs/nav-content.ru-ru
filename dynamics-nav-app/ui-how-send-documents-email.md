<span data-ttu-id="02930-105">Для быстрой передачи содержимого деловых документов вашим бизнес-партнерам, например информации о платеже в документах продажи клиентам, можно использовать функцию макетов отчетов, чтобы определять зависящее от вида документа содержимое, которое автоматически вставляется в текст сообщения электронной почты.</span><span class="sxs-lookup"><span data-stu-id="02930-105">To communicate the contents of business documents quickly to your business partners, such as the payment information on sales documents to customers, you can use the Report Layout feature to define document-specific content that gets inserted in email bodies automatically.</span></span> <span data-ttu-id="02930-106">Дополнительные сведения см. в разделе [Управление макетами отчетов и документов](ui-manage-report-layouts.md).</span><span class="sxs-lookup"><span data-stu-id="02930-106">For more information, see [Managing Report and Document Layouts](ui-manage-report-layouts.md).</span></span>

<span data-ttu-id="02930-107">Чтобы включить сообщения электронной почты в [!INCLUDE[d365fin](includes/d365fin_md.md)], на главной странице запустите сопровождаемую настройку **Настройка электронной почты**.</span><span class="sxs-lookup"><span data-stu-id="02930-107">To enable emails from within [!INCLUDE[d365fin](includes/d365fin_md.md)], start the **Set Up Email** assisted setup on the Home page.</span></span>

<span data-ttu-id="02930-108">Отправлять по электронной почте можно практически все типы документов в виде вложений непосредственно из окна, содержащего документ.</span><span class="sxs-lookup"><span data-stu-id="02930-108">You can email practically all document types as attachments to email messages directly from the window that shows the document.</span></span> <span data-ttu-id="02930-109">В дополнение к вложениям вы можете включать в текст сообщения электронной почты основную информацию из документа после стандартного текста с обращением к получателю и вводной информации о документе.</span><span class="sxs-lookup"><span data-stu-id="02930-109">In addition to the attachment, you can set up document-specific email bodies with core information from the document preceded by standard text that greets the mail recipient and introduces the document in question.</span></span> <span data-ttu-id="02930-110">Чтобы предложить клиентам оплатить покупку с помощью электронной службы платежей, например PayPal, вы можете включить в тело сообщения гиперссылку на PayPal.</span><span class="sxs-lookup"><span data-stu-id="02930-110">To offer your customers to pay for sales electronically using a payment service, such as PayPal, you can also have the PayPal information and hyperlink inserted in the email body.</span></span>

<span data-ttu-id="02930-111">Из всех поддерживаемых документов отправка электронной почты инициируется действием **Отправить** для учтенных документов и **Учесть и отправить** для неучтенных документов.</span><span class="sxs-lookup"><span data-stu-id="02930-111">From all supported documents, you initiate emailing by choosing the **Send** action, on posted documents, or the **Post and Send** action, on non-posted documents.</span></span>

<span data-ttu-id="02930-112">Если в поле **Эл. почта** окна **Кому отправить документ** установлено значение **Да (запросить настройки)**, то откроется окно **Отправить сообщение эл. почты** с заполненным полем **Кому:** и документом, приложенным в виде PDF-файла.</span><span class="sxs-lookup"><span data-stu-id="02930-112">If the **Email** field in the **Send Document to** window is set to **Yes (Prompt for Settings)**, then the **Send Email** window opens prefilled with the contact person in the **To:** field and the document attached as a PDF file.</span></span> <span data-ttu-id="02930-113">В поле **Содержание** вы можете либо ввести текст вручную или заполнить его настроенным текстом, связанным с типом документа.</span><span class="sxs-lookup"><span data-stu-id="02930-113">In the **Body** field, you can either enter text manually or you can have the field filled with a document-specific email body that you have set up.</span></span>

<span data-ttu-id="02930-114">Ниже показано, как настроить отчет **Продажи - счет**, чтобы использовать готовый текст при отправке по электронной почте учтенных счетов.</span><span class="sxs-lookup"><span data-stu-id="02930-114">The following procedure describes how to set the **Sales - Invoice** report up to be used for document-specific email bodies when you email posted sales invoices.</span></span>

## <a name="to-set-up-a-document-specific-email-body-for-sales-invoices"></a><span data-ttu-id="02930-115">Настройка содержания сообщения электронной почты для счетов продажи</span><span class="sxs-lookup"><span data-stu-id="02930-115">To set up a document-specific email body for sales invoices</span></span>
1. <span data-ttu-id="02930-116">Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Выбор отчета - продажи**, а затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="02930-116">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Selections Sales**, and then choose the related link.</span></span>
2. <span data-ttu-id="02930-117">В окне **Выбор отчета - продажи** в поле **Использование** выберите **Счет**.</span><span class="sxs-lookup"><span data-stu-id="02930-117">In the **Report Selection - Sales** window, in the **Usage** field, select **Invoice**.</span></span>
3. <span data-ttu-id="02930-118">В новой строке в поле **Код отчета** выберите, например, стандартный отчет 1306.</span><span class="sxs-lookup"><span data-stu-id="02930-118">On a new line, in the **Report ID** field, select, for example, standard report 1306.</span></span>
4. <span data-ttu-id="02930-119">Установите флажок **Использовать для содержания сообщения электронной почты**.</span><span class="sxs-lookup"><span data-stu-id="02930-119">Select the **Use for Email Body** check box.</span></span>
5. <span data-ttu-id="02930-120">Выберите поле **Код макета содержания сообщения электронной почты**, а затем выберите макет из раскрывающегося списка.</span><span class="sxs-lookup"><span data-stu-id="02930-120">Choose the **Email Body Layout Code** field, and then select a layout from the drop-down list.</span></span>

    <span data-ttu-id="02930-121">Макеты отчетов определяют стиль и содержимое сообщений электронной почты, включая стандартный текст, который предшествует информации из основного документа в теле сообщения электронной почты.</span><span class="sxs-lookup"><span data-stu-id="02930-121">Report layouts define both the style and the content of the email body, including the standard text that precedes the core document information in the email body.</span></span> <span data-ttu-id="02930-122">Можно просмотреть все доступные макеты отчетов, если нажать кнопку **Выбор из полного списка** в раскрывающемся списке.</span><span class="sxs-lookup"><span data-stu-id="02930-122">You can see all available report layouts if you choose the **Select from full list** button in the drop-down list.</span></span>
6. <span data-ttu-id="02930-123">Чтобы просмотреть или изменить макет содержания сообщения электронной почты, выберите макет в окне **Пользовательские макеты отчетов** выберите действие **Изменить макет**.</span><span class="sxs-lookup"><span data-stu-id="02930-123">To view or edit the layout that the email body is based on, select the layout in the **Custom Report Layouts** window, and then choose the **Edit Layout** action.</span></span>
7. <span data-ttu-id="02930-124">Если вы хотите предложить клиентам оплатить покупку с помощью электронной службы платежей, вы можете настроить электронную службу платежей, например PayPal, и включить в тело сообщения гиперссылку на PayPal.</span><span class="sxs-lookup"><span data-stu-id="02930-124">If you want to offer customers to pay for sales electronically, you can set up the related payment service, such as PayPal, and then have the PayPal information and hyperlink inserted in the email body as well.</span></span> <span data-ttu-id="02930-125">Дополнительные сведения см. в разделе [Практическое руководство. Включение платежей клиентов через PayPal](sales-how-enable-payment-service-extensions.md).</span><span class="sxs-lookup"><span data-stu-id="02930-125">For more information, see [How to: Enable Customer Payments Through PayPal](sales-how-enable-payment-service-extensions.md).</span></span>
8. <span data-ttu-id="02930-126">Нажмите кнопку **ОК**.</span><span class="sxs-lookup"><span data-stu-id="02930-126">Choose the **OK** button.</span></span>

<span data-ttu-id="02930-127">Теперь, когда вы выбираете, например, действие **Отправить** в окне **Учтенный счет продажи**, содержание сообщения электронной почты будет включать информацию из отчета 1306, перед которой будет расположен стандартный текст в соответствии с макетом отчета, выбранном на шаге 5.</span><span class="sxs-lookup"><span data-stu-id="02930-127">Now, when you choose, for example, the **Send** action in the **Posted Sales Invoice** window, the email body will contain the document information of report 1306 preceded by styled standard text according to the report layout that you selected in step 5.</span></span>

<span data-ttu-id="02930-128">Следующая процедура описывает, как отправить учтенный счет продажи в виде сообщения электронной почты с документом, вложенным в виде файла PDF, и с телом сообщения на базе этого документа.</span><span class="sxs-lookup"><span data-stu-id="02930-128">The following procedure describes how to send a posted sales invoice as an email message with the document attached as a PDF file and with a document-specific email body.</span></span>

# <a name="how-to-set-up-and-report-intrastat"></a>Практическое руководство. Настройка и подача отчетности Интрастат
Все организации в странах ЕС должны подавать отчет о торговых операциях с другими странами/регионами Евросоюза. Необходимо подавать ежемесячный отчет о перемещении товаров в статистическое бюро, а также отчет в налоговые органы вашей страны/региона. Эти функции называются "Отчеты Интрастат". Заполнение периодических отчетов Интрастат осуществляется на странице **Журнал Интрастат**.  
  
## <a name="required-and-optional-setups"></a>Обязательные и необязательные настройки
Перед началом использования журнала Интрастат для подачи информации Интрастат необходимо выполнить несколько настроек:  
  
* **Шаблоны журнала Интрастат**: необходимо настроить шаблоны и раздела журнала Интрастат, которые будут использоваться. Поскольку отчет Интрастат подается ежемесячно, необходимо создать 12 разделов журнала Интрастат на базе одного и того же шаблона.  
* **Товарные коды**: таможенные и налоговые органы определяют числовые коды классификации товаров и услуг. Вы указываете эти коды на товарах.
* **Коды характера транзакции**: в разных странах и регионах имеются различные коды для типов транзакций Интрастат, таких как обычная купля и продажа, замена возвращенных товаров и замена невозвращенных товаров. Настройте все коды, которые применяются в вашей стране или регионе. Эти коды используются в документах покупок и продаж, а также при обработке возвратов.  
* **Методы перевозки**: предусмотрено семь однозначных кодов для методов перевозки Интрастат. **1** для морских перевозок, **2** для железнодорожных перевозок, **3** для автомобильных перевозок, **4** для авиационных перевозок, **5** для отправке по почте, **7** для фиксированных установок и **9** для перевозок своим ходом (например, транспортировка автомобиля своим ходом). Для Dynamics NAV эти коды не требуются, однако рекомендуется, чтобы описания имели аналогичное значение.  

При желании можно также настроить:

* **Спецификации транзакции**: с их помощью можно дополнять описания из типов транзакций.  
* **Области**: с их помощью можно указывать дополнительную информацию о странах и регионах.  
* **Пункты ввоза/вывоза**: с их помощью можно указывать склады, на которые отправляются товары в другие страны или на которых получаются товары из других стран. Примером пункта ввоза или вывоза может служить аэропорт Хитроу. Пункт ввоза или вывоза указывается в документах продаж и покупок на экспресс-вкладке **Внешняя торговля**. Эта информация также копируется из записей о товарах при создании журнала Интрастат.  

### <a name="to-set-up-intrastat-templates-and-batches"></a>Настройка шаблонов и разделов Интрастат
Пакетное задание Интрастат включает только операции товаров, но не операции главной книги. При наличии операций главной книги, которые должны быть включены в отчетность Интрастат, необходимо ввести их вручную. Например, если вы приобрели компьютер в другой стране или /регионе ЕС, этот компьютер не включается в состав запасов, но учитывается в счете главной книги. Необходимо вручную ввести операции такого типа в журнал Интрастат.  
  
Операции можно экспортировать в файл, который можно отправить в органы Интрастат. Отчет можно также напечатать, вручную ввести сведения в формы из ответственных органов, и затем подать информацию.

>  [!Note]
> Рекомендуется настроить раздел журнала Интрастат для каждого месяца.  
  
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Шаблоны журналов Интрастат**, затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]. Создайте шаблон для каждой формы Интрастат, которую вы используйте.  
3. Для создания разделов выберите вкладку **Навигация**, затем выберите **Разделы**.  
4. Заполните соответствующим образом поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]. Создайте шаблон для каждой формы Интрастат, которую вы используйте.  

> [!Note]
> В поле **Статистика - период** введите статистический период в виде четырехзначного числа, в котором первые две цифры указывают на год, а следующие две — на месяц. Например, введите 1706 для июня 2017 г.

### <a name="to-set-up-commodity-codes"></a>Настройка товарных кодов
Все покупаемые и продаваемые вами товары должны иметь товарный код.  
  
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Товарные коды**, затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]  
3. Чтобы назначить товарный код товару, перейдите на страницу **Карточка товара**, разверните экспресс-вкладку **Затраты и учет**, затем введите код в поле **Товарный код**.   

### <a name="to-set-up-transaction-nature-codes"></a>Настройка кодов характера транзакции
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Коды характера транзакции**, затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]  

> [!Tip]
> Если вы часто используете конкретный код характера транзакции, можно назначить его кодом по умолчанию. Чтобы это сделать, перейдите на страницу **Настройка Интрастат**, затем выберите код. 

### <a name="to-set-up-transport-methods"></a>Настройка методов перевозки
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Методы перевозки**, затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]  

## <a name="to-report-intrastat"></a>Для подачи отчета Интрастат
После заполнения журнала Интрастат можно напечатать отчет **Контрольный список**, чтобы проверить правильность всей информации в журнале. Затем можно распечатать отчет Интрастат в виде формы или создать файл для отправки в налоговый орган вашей страны или региона.  

### <a name="to-fill-in-intrastat-journals"></a>Заполнение журналов Интрастат  
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журнал Интрастат**, затем выберите связанную ссылку.  
2. На странице **Журнал Интрастат** в поле **Название пакета** выберите нужный пакета журнала, затем нажмите кнопку **OK**.  
3. Выберите действие **Предложить строки**. Поля **Дата начала** и **Дата окончания** уже содержат даты, указанные в разделе журнала для периода статистики.  
4. В поле **Рег. себестоимости (%)** можно указать процент для покрытия транспортных расходов и расходов на страхование. Если указать процент, содержимое поля **Статистическое значение** в журнале будет пропорционально выше.  
5. Нажмите **ОК** для запуска пакетного задания.  
  
При выполнении этого пакетного задания выполняется поиск всех операций с товаром в течение статистического периода и такие операции вставляются в журнал Интрастат в виде строк. При необходимости можно отредактировать строки.  
  
> [!IMPORTANT]  
>  Пакетное задание находит только те операции, которые содержат код страны или региона, для которых на странице **Страны/регионы** был введен код Интрастат. Поэтому необходимо указывать коды Интрастат стран/регионов, по которым будет выполняться пакетное задание.  

### <a name="how-to-report-intrastat-on-a-form-or-a-file"></a>Практическое руководство. Предоставление отчета Интрастат на бланке или в файле
Для получения информации, требуемой для заполнения формы Интрастат, предоставляемой в органы статистики, следует распечатать отчет **Интрастат - форма**. Перед этим следует подготовить журнал «Интрастат» и заполнить его. Если имеются транзакции по покупкам и по продажам, следует заполнить отдельную форму для каждого типа, т. е. придется распечатать отчет дважды.  
  
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журналы Интрастат**, затем выберите связанную ссылку.  
2. На странице **Журнал Интрастат** выберите соответствующий раздел журнала в поле **Код раздела**.  
3. Если это еще не сделано, заполните журнал вручную или выберите **Предложить строки**.  
4. Выберите действие **Печать журнала Интрастат**.  
5. На экспресс-вкладке **Журнал Интрастат - строка** добавьте фильтр **Тип** и укажите для него значение **Накладная** или **Расх. накладная**.  
6. Выберите **Отправить** для печати отчета.  

### <a name="how-to-report-intrastat-in-a-file"></a>Практическое руководство. Передача отчетов Интрастат в виде файла
Отчет Интрастат может быть представлен в виде файла. Перед созданием файла можно распечатать контрольную таблицу, содержащую ту же информацию, что и в файле.  
  
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журнал Интрастат**, затем выберите связанную ссылку.  
2. В окне **Журнал Интрастат** выберите соответствующий раздел журнала в поле **Код раздела**.  
3. Если это еще не сделано, заполните журнал вручную или выбрав **Предложить строки**.  
4. Выберите действие **Создать файл**.  
5. В окне пакетного задания нажмите кнопку **OK**.  
6. Выберите **Сохранить**.  
7. Перейдите в расположение, в котором требуется сохранить файл, ведите имя файла, затем выберите **Сохранить**. 

## <a name="how-to-reorganize-intrastat-journals"></a>Практическое руководство. Перестройка журналов Интрастат
Поскольку отчет Интрастат должен предоставляться каждый месяц и для каждого отчета следует создавать новый раздел журнала, со временем таких разделов журнала будет много. Строки журнала автоматически не удаляются. Периодически может возникать необходимость перестройки кодов разделов журнала. Это делается путем удаления разделов журнала, в которых больше нет необходимости. Строки таких разделов журнала также удаляются.  
  
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журналы Интрастат**, затем выберите связанную ссылку.  
2. Для просмотра параметров выберите поле **Код раздела**.  
3. Выберите разделы журнала, которые требуется удалить, затем выберите **Удалить**.

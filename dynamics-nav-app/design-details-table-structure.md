# <a name="design-details-table-structure"></a>Сведения о проектировании: структура таблицы
Чтобы понять, как переконструировано хранение и учет операций измерения, важно понять структуру таблицы.  

## <a name="new-tables"></a>Новые таблицы  
 Созданы три новые таблицы для управления операциями с набором измерений.  

### <a name="table-480-dimension-set-entry"></a>Таблица 480 "Запись набора измерений"  
 Таблица 480 **Запись набора измерений** новая. Эту таблицу невозможно изменить. После записи данных в таблицу их невозможно удалить или изменить. Для удаления данных требуется выполнить проверку по всем вхождениям кода набора измерений во всей базе данных, включая решения партнеров.  

|Номер поля|Имя поля|Тип данных|Комментарий|  
|---------------|----------------|---------------|-------------|  
|1|**Код**|Целое|>зарезервировано 0,0 для получения пустого набора измерений. Ссылается на поле 3 в таблице 481.|  
|2|**Код Измерения:**|Код 20|Отношение таблицы с таблицей 348.|  
|3|**Код Значения Измерения**|Код 20|Отношение таблицы с таблицей 349.|  
|4|**ИД значения измерения**|Целое|Ссылается на поле 12 в таблице 349. Это вторичный ключ, который используется при переходе по таблице 481.|  
|5|**Название измерения**|Текст 30|CalcField. Выбирается в таблице 348.|  
|6|**Имя значения измерения**|Текст 30|CalcField. Выбирается в таблице 349.|  

#### <a name="table-481-dimension-set-tree-node"></a>Таблица 481 "Узел дерева набора измерений"  
 Таблица 481 **Узел дерева набора измерений** новая. Эту таблицу невозможно изменить. Она используется для поиска набора измерений. Если набор измерений не найден, создается новый набор.  

|Номер поля|Имя поля|Тип данных|Комментарий|  
|---------------|----------------|---------------|-------------|  
|1|**Код родительского набора измерений**|Целое|0 для узла верхнего уровня.|  
|2|**ИД значения измерения**|Целое|Отношение таблицы с полем 12 в таблице 349.|  
|3|**Код набора измерений**|Целое|AutoIncrement. Используется в поле 1 таблицы 480.|  
|4|**Используется**|Логическое|Значение false, если не используется.|  

##### <a name="table-482-reclas-dimension-set-buffer"></a>Таблица 482 "Буфер реклассиф. набора измерений"  
 Таблица 482 **Буфер реклассиф. набора измерений** новая. Таблица используется для редактирования ИД набора измерений. Это необходимо при изменении кода значения измерения и создании нового кода значения измерения, например в таблице **Журнал реклассификации товаров**.  

|Номер поля|Имя поля|Тип данных|Комментарий|  
|---------------|----------------|---------------|-------------|  
|1|**Код Измерения:**|Код 20|Отношение таблицы с таблицей 348.|  
|2|**Код Значения Измерения**|Код 20|Отношение таблицы с таблицей 349.|  
|3|**ИД значения измерения**|Целое|Ссылается на поле 12 в таблице 349.|  
|4|**Измерение Новый Код Значения**|Код 20|Отношение таблицы с таблицей 349.|  
|5|**Новый ИД значения измерения**|Целое|Ссылается на поле 12 в таблице 349.|  
|6|**Название измерения**|Текст 30|CalcField. Выбирается в таблице 348.|  
|7|**Имя значения измерения**|Текст 30|CalcField. Выбирается в таблице 349.|  
|8|**Новое имя значения измерения**|Текст 30|CalcField. Выбирается в таблице 349.|  

## <a name="modified-tables"></a>Измененные таблицы  
 Все таблицы транзакций и бюджета были изменены для управления операциями набора измерений.

|Номер поля|Имя поля|Тип данных|Комментарий|  
|---------------|----------------|---------------|-------------|  
|480|**Код набора измерений**|Целое|Ссылается на поле 1 в таблице 480.|  

#### <a name="changes-to-table-83-item-journal-line"></a>Изменения в таблице 83 "Строка журнала товаров"  
 Два новых поля добавлены в таблицу 83 **Строка журнала товаров**.  

|Номер поля|Имя поля|Тип данных|Комментарий|  
|---------------|----------------|---------------|-------------|  
|480|**Код набора измерений**|Целое|Ссылается на поле 1 в таблице 480.|  
|481|**Новый код набора измерений**|Целое|Ссылается на поле 1 в таблице 480.|  

##### <a name="changes-to-table-349-dimension-value"></a>Изменения в таблице 349 "Значение измерения"  
 Добавлено новое поле в таблицу 349 **Значение измерения**.  

|Номер поля|Имя поля|Тип данных|Комментарий|  
|---------------|----------------|---------------|-------------|  
|12|**ИД значения измерения**|Целое|AutoIncrement. Используется для ссылок в таблицах 480 и 481.|
